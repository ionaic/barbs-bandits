Clears the result image to a color (black is default). 

Renders the background of the element, namely element contents. For generic Elements, it blits a solid color (black) image to the element's result image. For content elements (\hyperlink{class_text_element}{Text\-Element} and \hyperlink{class_image_element}{Image\-Element}) it will blit the stored image (for image elements) or resulting image from rendering the text (for text elements) before rendering the children.
