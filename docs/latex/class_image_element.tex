\subsection{Detailed Description}
An element which simply contains an image and has children. 

Its main purpose is to provide constructors and to override \hyperlink{class_element_a7a110bce4630cd7b35b1c2d401774598}{Element\-::clear\-Result()} to fill the background with the contained image instead of a plain color.
